# Emberpath

Mobile-first top-down exploration game with story scenes, inspired by Pilgrim's Progress, using AI-generated art (Midjourney + Flow).

## Getting started

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (default: `http://localhost:5173`).

## Controls

- **Desktop:** W/A/S/D to move up/left/down/right
- **Mobile:** Touch and drag anywhere to activate the virtual joystick

## Build

```bash
npm run build
```

Produces a deployable `dist/` directory.
